# 修改计划：支持手动设定启动端口

**日期**：2026-06-07

## 目标

让服务器支持通过环境变量或命令行参数手动指定端口，而非硬编码 3000。

## 方案

优先级：命令行参数 > 环境变量 > 配置文件默认值

```
node server.js --port 8080    # 命令行参数（最高优先）
PORT=8080 node server.js      # 环境变量
node server.js                # 使用 config/default.js 中的默认值 3000
```

## 修改文件

| 文件 | 变更 |
|------|------|
| `config/default.js` | 默认端口改为读取 `process.env.PORT` |
| `server.js` | 解析 `--port` 命令行参数，优先级最高 |

## 验证

```bash
node server.js --port 8080        # 监听 8080
PORT=4000 node server.js          # 监听 4000
node server.js                    # 监听 3000（默认）
```
